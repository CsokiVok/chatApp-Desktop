# Chat Application - Frontend

Ez a chat alkalmazás frontend része, amely React technológiára épül. Az alkalmazás biztosítja a felhasználói felületet, valós idejű üzenetküldést és adminisztrációs funkciókat.

## Főbb funkciók

- **Felhasználói regisztráció és bejelentkezés**: A felhasználók regisztrálhatnak, bejelentkezhetnek, és profiljukat frissíthetik.
- **Valós idejű üzenetküldés**: Szöveges és képes üzenetek küldése és fogadása.
- **Felhasználói státusz**: Online/offline státusz valós idejű frissítése.

## Technológiák

- **React**: A felhasználói felület megvalósításához.
- **React Router**: Az oldalak közötti navigációhoz.
- **Zustand**: Állapotkezeléshez.
- **Tailwind CSS és DaisyUI**: A modern és reszponzív dizájn érdekében.
- **Vite**: Gyors fejlesztői környezet.

## Mappastruktúra
components/: Újrafelhasználható React komponensek.
pages/: Oldalak, például Home, Login, SignUp, Admin.
store/: Zustand állapotkezelés.
lib/: Segédfüggvények, például Axios konfiguráció.
